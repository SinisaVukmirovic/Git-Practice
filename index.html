Git Practice

Git Commands

To initializing a local git repository

git init

To add name and email to git

git config --global user.name 'your name'

git config --global user.email 'your email'

To add a file to git staging area

git add filename.ext

To add all files of same type (ex: .html files) to git staging area

git add *.html

To add everything / all files

git add .

To check the status of files in git repository

git status

To remove a file from git staging area

git rm --cached filename.ext

To commit files in git repo

git commit -m 'Commiting message'

To see a list of all commits

git log --oneline

To see all commits with more details

git log

To create a new branch

git branch branchname

To switch to branch

git checkout branchname

To merge git branches

From branch we want to merge to

git merge branchname

To push to a remote repository on GitHub

After creating a GitHub repo

git remote add origin https://...

then

git push -u origin master
